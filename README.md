## Infos
**Server-IP:** `homiecraft.example.de`  
**Version:** 1.20  
**Plugins / Mods:** EssentialsX, WorldEdit, Dynmap  

**Was du auf dem Server machen kannst:**  
- Baue dein eigenes Haus oder Stadtviertel  
- Erkunde die Welt und finde seltene Ressourcen  
- Nimm an PvP- oder Bau-Events teil  
- Nutze Wirtschaftssysteme oder Shops  
- Spiele zusammen mit Freunden im Team  
- Zeige deine Kreativität auf Community-Projekten

## Commands
| Befehl | Beschreibung |
|--------|-------------|
| `/spawn` | Teleportiert dich zum Spawnpunkt |
| `/home` | Teleportiert dich zu deinem eigenen Haus |
| `/warp <Ort>` | Teleportiert zu einem definierten Warp-Punkt |
| `/tpa <Spieler>` | Teleportiere dich zu einem Spieler |
| `/msg <Spieler> <Nachricht>` | Sende private Nachrichten |

---

## Hilfe
**FAQ / Tipps:**
1. Wie erstelle ich ein Haus? → Nutze `/sethome` und baue in deiner Zone.  
2. Wie trete ich Events bei? → Prüfe die Event-Announcements im News-Ordner.  
3. Probleme mit Plugins? → Kontaktiere ein Teammitglied über `/msg <Admin>`
