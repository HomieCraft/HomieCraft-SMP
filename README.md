## Commands
| Befehl | Beschreibung |
|--------|-------------|
| `/spawn` | Teleportiert dich zum Spawnpunkt |
| `/home` | Teleportiert dich zu deinem eigenen Haus |
| `/tpa <Spieler>` | Teleportiere dich zu einem Spieler |
| `/msg <Spieler> <Nachricht>` | Sende private Nachrichten |

## Hilfe
**FAQ / Tipps:**
