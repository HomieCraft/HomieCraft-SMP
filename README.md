## Commands
| Befehl | Beschreibung |
|--------|-------------|
| `/spawn` | Teleportiert dich zum Spawnpunkt |
| `/home` | Teleportiert dich zu deinem eigenen Haus |
| `/tpa <Spieler>` | Teleportiere dich zu einem Spieler |
| `/msg <Spieler> <Nachricht>` | Sende private Nachrichten |

## Hilfe
**FAQ / Tipps:**
1. Wie erstelle ich ein Haus? → Nutze `/sethome` und baue in deiner Zone.  
2. Wie trete ich Events bei? → Prüfe die Event-Announcements im News-Ordner.  
3. Probleme mit Plugins? → Kontaktiere ein Teammitglied über `/bugreport`
